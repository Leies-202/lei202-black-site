個人情報根掘り葉掘り聞かれたり、認証が面倒だったり、できればそもそも広告消しときたいし(あるのもまた味になりそうですが)

3つぐらいしか無料レンタルサーバー知らんし...。

というかGitHubが手っ取り早すぎた。
